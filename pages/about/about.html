About Our Project

This is a Final Year Project (FYP) developed by a team of dedicated students. Our project, PentestTools, is a web-based vulnerability scanner designed to help identify security weaknesses in web applications. As students passionate about cybersecurity, we've combined our academic knowledge with practical implementation to create this platform.

Project Overview

Our vulnerability scanner is designed to detect common security issues in web applications, including SQL injection, XSS (Cross-Site Scripting), and other potential vulnerabilities. The project aims to provide a user-friendly interface for conducting security assessments while learning about web security best practices.

Our Goals

As students, our primary goals are to:

  • Learn and implement web security concepts
  • Create a practical tool that can help identify security vulnerabilities
  • Gain hands-on experience in cybersecurity
  • Develop a user-friendly interface for security testing

Features

Our scanner includes features such as:

  • Basic vulnerability scanning
  • Detailed scan reports
  • User-friendly interface
  • Security recommendations
